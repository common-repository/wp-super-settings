== Description ==

WP Super Settings allows you to do various generally required settings in WordPress which are otherwise not available directly. 

Major settings/features in WP Super Settings 1.0.0 includes:

* Enable/Disable Autosave
* Enable/Disable Revisions
* Show/Hide Admin Bar
* ShortCode for embedding other post(s) in current post

== Installation ==

1. Install automatically through the `Plugins`, `Add New` menu in WordPress, or upload the `wp-super-settings` folder to the `/wp-content/plugins/` directory. 
2. Activate the plugin through the `Plugins` menu in WordPress. 
3. Updates process is automatic. Click on "Upgrade Automatically" if prompted from the WordPress admin menu. During manually upgrade, simply deactivate, uninstall, and repeat the installation steps.

== Frequently Asked Questions ==

= How to embed other post in current post? =

Simply use the [Embed_Post post_id=000] shortcode. Replace 000 with required post ID.
